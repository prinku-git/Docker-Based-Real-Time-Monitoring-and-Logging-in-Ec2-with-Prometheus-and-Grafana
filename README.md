## Overview

This project implements a **real-time monitoring and centralized logging solution** on an **AWS EC2 (Amazon Linux)** instance using **Dockerized monitoring tools**.
It provides visibility into system performance, logs, and alerting for cloud infrastructure.

---

## Tech Stack

* **AWS EC2 (Amazon Linux)**
* **Docker**
* **Prometheus**
* **Node Exporter**
* **Grafana**
* **Loki**
* **Promtail**
* **Alertmanager**

---

## Outline

* Deployed all monitoring components as **Docker containers** on EC2
* Configured **Prometheus** to scrape system metrics via Node Exporter
* Set up **Loki + Promtail** for centralized log aggregation
* Integrated **Prometheus and Loki** as data sources in **Grafana**
* Configured **Alertmanager** for alert routing and notifications
* Verified metrics, logs, dashboards, and alert status

---

## Key Features

* Real-time system monitoring
* Centralized log management
* Grafana dashboards for metrics and logs
* Alerting framework for proactive monitoring
* Cloud-native, production-style setup
